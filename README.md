# Low Level Design (LLD) Practice 🚀

This repository contains my implementations of common **Low Level Design (LLD)** interview problems using **C++**.

The goal of this repo is to:

* Practice object-oriented design
* Improve system design thinking
* Prepare for software engineering interviews
* Build clean, extensible, and scalable designs

---

## 📌 Problems Covered

### 🟢 Easy

* Parking Lot System
* Tic Tac Toe
* Snake and Ladder
* ATM System

### 🟡 Medium (Upcoming)

* Vending Machine
* Elevator System
* Splitwise

### 🔴 Advanced (Upcoming)

* BookMyShow
* Notification System
* Rate Limiter

---

## 🧠 Design Principles Used

* Encapsulation
* Abstraction
* Separation of Concerns
* Clean Code Practices
* Extensible Design

---

## 🛠 Tech Stack

* Language: C++
* Concepts: OOP, STL, Design Patterns
